# TATEGAKI STUDIO

縦書き台本作成に特化した、軽量なWebエディタです。  
横書きのTXT台本を読み込み、A4横向きの縦書き原稿として整形し、PDF書き出しや編集データ保存ができます。

## Features

- A4横向き固定の縦書き原稿プレビュー
- TXTファイルのドラッグ＆ドロップ読み込み
- 自動ページ分割
- 全ページ通しの行番号表示
- 文字サイズ、行間、余白、フォント変更
- 選択範囲への太字、文字色、部分文字サイズ指定
- 縦書き用の句読点、括弧、三点リーダー表示調整
- 執筆モードによる行単位編集
- `Ctrl+Enter` による空行保持
- `Tab` による2文字下げ
- `Shift+Tab` による2文字下げ解除
- PDF書き出し
- JSON形式の編集データ書き出し / 読み込み
- PC / iPad / スマホ対応のレスポンシブUI

## Usage

### TXTを読み込む

左カラムの「ここにtxtをドロップ」に `.txt` ファイルをドラッグ＆ドロップします。  
読み込まれた文章は、自動でA4横向きの縦書き原稿に流し込まれます。

### 執筆モード

上部の「執筆モード」を押すと、原稿上の行をクリックして編集できます。

- `Ctrl+Enter`：空行を保持して改行
- `Tab`：2文字下げ
- `Shift+Tab`：2文字下げ解除

編集後は「反映」を押すと、本文に反映されて再レイアウトされます。

### PDF書き出し

上部の「PDF書き出し」を押すと、ブラウザの印刷ダイアログが開きます。  
保存先で「PDFに保存」を選ぶことで、A4横向きの縦書きPDFとして保存できます。

### 編集データの保存

左カラムの「ファイル保存」を押すと、JSONバックアップが表示されます。  
「全選択」を押して `Ctrl+C` でコピーし、メモ帳などに貼り付けて `.json` ファイルとして保存してください。

### 編集データの読み込み

左カラムの「ファイル読込」から、保存した `.json` ファイルを選択すると編集状態を復元できます。

html2canvas や jsPDF によるスクリーンショット変換は使用していません。
